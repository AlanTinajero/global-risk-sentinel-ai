# 🌍 Global Risk Sentinel AI

An AI-powered global intelligence platform that monitors real-time risk events across the world using OSINT data.

---

## 🚀 Features

- 🌍 Real-time global risk map
- 🔥 Heatmap of dangerous zones
- 🧠 AI pattern detection (crime / risk clustering)
- 📡 Live OSINT data (GDELT integration)
- 🌐 Multi-language interface (EN / ES)
- 📱 Alert system (ready for mobile notifications)

---

## 🧠 Tech Stack

- Python
- Streamlit
- Scikit-learn (Machine Learning)
- Folium (Maps)
- Google Maps API (Geolocation)
- OSINT Data (GDELT)

---

## ⚡ Use Cases

- Cybersecurity monitoring
- OSINT investigations
- Risk intelligence
- Crisis detection
- Public safety awareness
